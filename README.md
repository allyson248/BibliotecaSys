# BibliotecaSys

Sistema de gerenciamento de biblioteca desenvolvido para estudos da disciplina de Full Stack II.

## Tecnologias

- Node.js
- Express
- React
- MySQL
- Git
- GitHub

## Funcionalidades

- Cadastro de usuários
- Cadastro de leitores
- Cadastro de autores
- Cadastro de categorias
- Cadastro de livros
- Controle de empréstimos
- Login de administradores e bibliotecários
